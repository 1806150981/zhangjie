## 介绍

基础数据类型就是 生活中的量词 比如说 一块钱 200块钱 块可以理解为 int long 类型 

1. 整型：byte、short、int、long
2. 浮点型：float、double
3. 字符型：char
4. 布尔型：boolean

## byte short int long 

byte:一个字节最小单位 

short:整形 1,2,3 默认值 0 定义  short a = 1

int:整形 12 3 默认值0 定义 int a = 1

long ：整形 long a = 1L

## float ,double

定义带小数点时候用 float 常用

```java
        float a = 1;
        float b = 1.0f;
        double c = 1.0;
        double e = 3.5;
```

## char

一个字符串

```java
char a = 'a';
```

## boolean

```java
boolean a = true;
```
